## Important Notes

1. **Copyright & Permissions:**
   - Ensure all images are properly licensed
   - Obtain permissions when using actual brand imagery
   - Use royalty-free stock photos or original photography

2. **Cultural Sensitivity:**
   - Ensure images reflect Sri Lankan diversity
   - Be mindful of religious and cultural considerations
   - Use professional, business-appropriate imagery

3. **Image Specifications:**
   - Maintain current image dimensions for consistency
   - Optimize for web (compress without quality loss)
   - Use WebP format where possible for better performance
   - Provide JPG/PNG fallbacks for older browsers

4. **Where to Source Images:**
   - Original photography from Sri Lankan locations
   - Licensed stock photos featuring Sri Lankan subjects
   - Brand-provided imagery (with permission)
   - Pexels, Unsplash (filter for Sri Lankan context)

5. **SEO Considerations:**
   - Update alt text to include "Sri Lanka", "Colombo", etc.
   - Use descriptive filenames (e.g., "odel-fashion-colombo.jpg")
   - Include location-specific keywords
